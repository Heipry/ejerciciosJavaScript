Ejercicio D-4

Armando una lista de super

  1. Crea un array vacio que se llame listaDeSuper.
  2. Piensa como le puedes pedir al usuario para que ingrese al menos 5 productos para llenar la lista de super.
  3. Muestra por la consola los productos introducidos por el usuario.

Forma de resolverlo:

Creo el array vacío que se llame listaDeSuper.

let listaDeSuper = [];

Con un prompt pido al usuario para que ingrese al menos 5 productos para llenar la lista de super.

let listaUsuario = prompt("Por favor ingrese 5 productos para armar la lista del supermercado, separe cada producto con una coma."); listaDeSuper = listaUsuario.split(",");

muestro por consola los productos en el array

console.log(listaDeSuper); Volver al inicio
