Ejercicio 35

Vamos a trabajar con la navegación del historial del navegador en JavaScript. En este ejercicio, crearás dos botones para navegar hacia atrás y hacia adelante en el historial.

Lo resolví del siguiente modo:

const btnBack = document.getElementById("btnBack");
const btnForward = document.getElementById("btnForward");

// Event listeners para los botones
btnBack.addEventListener("click", () => {
    console.log("Navegando al historial anterior...");
    history.back();
});

btnForward.addEventListener("click", () => {
    console.log("Navegando al siguiente elemento del historial...");
    history.forward();
});
