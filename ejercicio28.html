Ejercicio 28

Contador de click

  1. Crea un botón en HTML qué diga "clickeame"
  2. Crea un función contadorDeClick() y programa que cada vez que se haga click se muestre por la consola el conteo de las veces que se ha realizado click en el botón.

Primer botón:

let cantidad = 0; function contadorDeClick() { cantidad+=1; console.info(cantidad); }

Segundo botón:

let veces = 0; function contadorClick() { veces+01; console.log(veces) } let btn = document.getElementById("btn2") btn.addEventListener('click',contadorClick);
